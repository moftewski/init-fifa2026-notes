# init-fifa2026-notes

这是一个用于归档和发布多个独立 HTML 页面的仓库。这里存放的页面以静态 HTML 为主，不绑定任何特定域名或网站，主要作为个人或团队在项目推进过程中的记录、笔记与资料展示使用。

## 项目简介

本仓库旨在整理和公开与 FIFA 2026 相关的各类信息页面。这些页面内容独立，互不依赖，适合直接通过浏览器打开查看。所有页面均为静态 HTML，不涉及服务器端逻辑，便于快速部署与长期存档。

## 目录结构说明

```
├── index.html          # 仓库首页（目录导航）
├── pages/              # 存放所有 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 静态资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：所有独立 HTML 页面均放置在此目录下，文件名以简洁的编号或简短描述命名。
- `assets/`：页面引用的公共资源，按类型分目录存放。
- `index.html`：提供页面列表与快速跳转。

## 页面归档说明

每个页面均为独立完整的 HTML 文件，可直接在浏览器中运行。归档内容涵盖但不限于：

- 赛事相关记录与笔记
- 信息整理与可视化展示
- 项目过程性文档与演示

所有页面均不包含外部跟踪、广告或第三方脚本，仅依赖仓库内资源。

## 维护说明

- 本仓库由项目参与人员不定期更新，新增页面会放置在 `pages/` 目录下，并在 `index.html` 中添加链接。
- 页面内容以事实性记录为主，不保证实时更新或涵盖所有方面。
- 欢迎提出 Issues 或 Pull Requests，但请注意本仓库不用于商业推广或具体网站引流。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接使用浏览器打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可将整个仓库部署到任意静态托管服务，作为简单的页面集合站点。
